# compas-acoustics-test

Toy project to test template for compas acoustics

## Installation

Stable releases can be installed from PyPI.

```bash
pip install compas_acoustics_test
```

To install the latest version for development, do:

```bash
git clone https://github.com//compas_acoustics_test.git
cd compas_acoustics_test
pip install -e ".[dev]"
```
